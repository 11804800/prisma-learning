
# Prisma Learning

It is rest api using postgressql as db and prisma as orm it only operates basic CRUD operations



## Features

- Relationship
- CRUD operations



## Installation

Install my-project with npm

```bash

  npm i prisma
  npx prisma 
  npx prisma init

```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/11804800/prisma-learning.git
```

Go to the project directory

```bash
  cd prisma-learning
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

To connect the prisma to pgadmin

```bash
   npx prisma migrate dev --name create_table_name
```


## Tech Stack

**Server:** Node, Express, Prisma ,@prisma/client, Postgressql
